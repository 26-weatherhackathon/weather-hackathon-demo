# 우리 마을 홍수 방재 게임

국토지리정보원 수치표고모형(DEM)으로 만든 지형 위에서 물이 어디로 흐르고 어디가 잠기는지
직접 보고 마을을 지키는 초등학생용 웹 기반 교육 게임. 개발 배경과 전체 기획은
[`docs/PLAN.md`](docs/PLAN.md)를 참고한다.

- **대상 학습자**: 초등학교 4~6학년 (구체적 조작기 — 타이머·추상 수치 없이 직접 조작하며 학습)
- **학습 목표**: DEM 색상 지형도를 보고 물이 어디로 흐르고 어디가 먼저 잠기는지 설명할 수 있다 /
  땅의 높낮이가 침수를 결정한다는 인과를 근거를 들어 말할 수 있다

## 현재 구현 범위 (최소 기능 프로토타입)

`docs/PLAN.md` 6.2장 "STEP 1 자유 탐색"과 "STEP 2~4 지역 선택·선택지·결과"를 구현했다.

**STEP 1 — 자유 탐색** (타이머 없이 자유롭게 켜고 끄며 반복 비교)

- **지형도 보기**: 목업 DEM 격자를 고도별 색상(갈색/연두/초록/파랑)으로 아이소메트릭 렌더링
- **물 흐름 보기**: 셀별 D8(8방향 최저점) 흐름 방향을 화살표로 표시
- **비가 오면?**: 강우량 기준 침수 판정(고도 < 기준선 + 강우량)에 따라 저지대에 물 차오름 표시

**STEP 2~4 — 지역 선택 · 선택지 · 결과**

- 지형도에서 칸을 클릭하면(지역 선택) 큰 버튼 4개(높이 올리기 / 모래주머니 / 대피소 / 모두 피난)가 뜬다
- 선택 즉시 그 지역의 DEM 침수 판정이 다시 계산되어 반영된다(PLAN.md 5.3 규칙)
  - 높이 올리기: 고도 +30m → 침수선 위로 올라가 안전
  - 모래주머니: 방벽 +10m → 부분 차단(약한 비는 막고 강한 비는 넘침)
  - 대피소: 건물은 침수되어도 인명은 안전
  - 모두 피난: 마을 사람들이 격자에서 가장 높은 셀로 이동(집 아이콘으로 표시)
- 결과는 안전(초록 체크) / 경고(주황 세모) / 위험(빨강 엑스) SVG 아이콘과 문구로 안내하고,
  "다시 해보기"로 같은 지역에 다른 선택지를 다시 시도할 수 있다

DEM 처리는 `lib/dem.ts`에 `elevationToColor`(고도→색상), `computeD8Flow`(물 흐름 방향),
`isFlooded`(침수 판정), `applyProtectionElevation`(선택지별 고도 보정), `describeOutcome`(결과 문구)
함수로 분리해 두었다. 지금은 `createMockDem()`이 만드는 가상 격자를 쓰지만, 추후 국토지리정보원
실제 DEM(GeoTIFF→JSON)으로 교체할 때 이 지점만 바꾸면 되도록 구성했다.

## 실행 방법

```bash
npm install
npm run dev
```

`http://localhost:3000`에서 확인한다. 별도 환경변수나 API 키는 아직 필요하지 않다.

## 배포 방법

Next.js 표준 앱이라 Vercel에서 별도 설정 없이(zero-config) 빌드된다.

- **빌드 명령어**: `npm run build` (Vercel이 자동 감지)
- 이 저장소는 이미 Vercel 프로젝트(GitHub 연동)에 연결되어 있어, `main`에 PR을 열면
  Vercel이 자동으로 Preview 배포 URL을 PR 체크에 생성한다.
- 동일하게 재배포하려면: 이 저장소를 Vercel에 Import → Framework Preset: Next.js →
  Root Directory: `.` (기본값) → Deploy.

## 외부 리소스 출처 및 라이선스

이번 프로토타입은 외부 이미지·폰트·아이콘·데이터셋을 사용하지 않았다(코드로만 구현).
사용 라이브러리는 모두 오픈소스 표준 스택이다: Next.js / React (MIT), Tailwind CSS (MIT).

|---|---|---|---|
| (없음) | — | — | 현재 버전은 코드로만 구현된 목업 데이터 사용 |

## 사용한 기상·지리 데이터 및 API

현재 버전은 실데이터를 연동하지 않은 **목업 DEM 격자**(`lib/dem.ts`의 `createMockDem`)로
동작한다. `docs/PLAN.md` 4.1장에 따라 다음 실데이터로 교체할 계획이다.

- 국토지리정보원 5m DEM (geoinfo.go.kr / data.go.kr, 공공데이터 CC-BY) — 지형 고도
- 기상청 강수강도(RN_INT, data.kma.go.kr) — 강우량 입력값
